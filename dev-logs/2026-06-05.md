## 解决的问题
- **ELECTRON_RUN_AS_NODE=1**：终端环境变量导致 Electron 当作纯 Node 运行
  - 解决：运行前 `unset ELECTRON_RUN_AS_NODE`
- **better-sqlite3 编译失败**：需要 Visual Studio C++ 环境
  - 解决：改用 sql.js（纯 JS 实现）
- **cheerio 版本兼容**：新版 cheerio 依赖 undici 7.x，与 Electron Node 版本不兼容
  - 解决：使用 cheerio@1.0.0-rc.12
- **ShuffleOutlined 图标不存在**：@ant-design/icons 中没有这个图标
  - 解决：改用 SwapOutlined
