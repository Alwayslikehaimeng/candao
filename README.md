# 喰导 (CanDao)

本地视频归档和管理软件，自动抓取网络信息建立媒体库。

## 功能特性

### 核心功能
- **视频导入**：支持拖拽文件、文件夹批量扫描
- **自动爬取**：AV（FANZA）、FC2 自动抓取元数据
- **手动编辑**：所有信息支持手动修改保存
- **直接播放**：调用系统默认播放器打开视频
- **随机播放**：随机挑选一部视频播放

### 分类管理
- **AV**：输入番号自动爬取封面、演员、标签等
- **FC2**：输入 FC2 代码抓取作品信息
- **其他**：自动提取视频基础信息，手动补充

### 搜索与筛选
- 按番号、演员、标签搜索
- 按分类、评分范围筛选
- 按添加时间、发行日期、评分排序

## 技术栈

| 组件 | 技术 |
|------|------|
| 桌面框架 | Electron |
| 前端 | React + TypeScript |
| UI 组件 | Ant Design（暗色主题） |
| 数据库 | SQLite (better-sqlite3) |
| 爬虫 | axios + cheerio |
| 视频信息 | ffprobe |

## UI 设计

深色模式（Dark Mode）+ 粉色/魅魔紫主题色

- 背景色：#1A1A2E
- 主色：#FF6B9D（粉色）、#9B59B6（魅魔紫）
- 魅魔纹装饰融入界面细节

## 安装与运行

```bash
# 安装依赖
npm install

# 开发模式运行
npm run dev

# 构建打包
npm run build:win
```

## 使用说明

1. **添加视频**：点击"添加视频"按钮，输入番号和视频文件路径
2. **批量导入**：点击"批量导入"，选择文件夹自动扫描
3. **搜索**：在搜索框输入番号、演员或标签
4. **播放**：点击视频卡片或详情页的"播放"按钮
5. **随机播放**：点击"随机播放"按钮随机打开一部视频
6. **编辑信息**：进入详情页修改信息后点击"保存"

## 代理配置

如需访问 FANZA 等需要日本 IP 的网站：

1. 点击设置按钮
2. 配置代理地址（HTTP/SOCKS5）
3. 启用代理
