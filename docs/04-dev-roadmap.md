# 开发路线图

## 原则
- 每次只做一个功能，做完测试通过再做下一个
- 每个功能完成后构建并验证
- 遇到问题先记录，不要绕过

## 阶段一：Bug 修复（当前）
- [x] **S1-1 悬停效果优化**
  - 标题悬停变粉色 + 展开两行
  - 验证：悬停时标题变色并显示完整内容

- [x] **S1-2 FANZA 链接修复**
  - 用 shell.openExternal 在浏览器打开
  - 验证：点击 FANZA 按钮能打开网页

- [x] **S1-3 删除清理**
  - 删除视频时同时删除下载的封面和示例图
  - 验证：删除后 covers/ 目录中对应文件也被删除
