# 喰导 (CanDao) — 产品需求文档

## 产品定位
本地视频归档和管理软件，自动抓取网络信息建立媒体库，解决找视频难、信息不全的问题。

## 目标用户
个人用户，硬盘中有大量杂乱视频文件，需要整理和快速查找。

## 核心功能

### 1. 视频导入
- 支持单个文件选择（系统文件对话框）
- 支持文件夹批量扫描
- 自动从文件名提取番号（AV/FC2 格式）
- 只记录文件路径，不移动/复制文件
- 删除时只删数据库记录和下载的图片，不删本地视频文件

### 2. 分类系统
- **AV**：番号格式如 JUR-258、SSNI-001
- **FC2**：格式如 FC2-PPV-1234567
- **其他**：所有不符合上述格式的视频

### 3. 自动信息抓取
- **AV 分类**：从 FANZA 抓取封面、示例图、标题、发行日期、片长、演员、导演、制造商、标签、评分
- **FC2 分类**：从 FC2 官网抓取标题、封面、卖家信息、标签
- 封面和示例图下载到本地存储
- 需要代理才能访问 FANZA（自动检测系统代理）

### 4. 手动编辑
- 所有字段支持手动修改保存
- 查看/编辑模式切换
- "其他"分类：自动提取视频基础信息（时长、分辨率）+ 手动填写

### 5. 播放功能
- 点击播放 → 调用系统默认播放器打开文件
- 随机播放按钮 → 纯随机挑选一部播放

### 6. 搜索与筛选
- 搜索框：按番号、演员、标签搜索
- 排序：添加日期、发行日期、评分
- 筛选：分类（AV/FC2/其他）

### 7. 批量管理
- 批量删除选中的视频记录和下载的图片

### 8. 榜单功能
- 爬取 DMM 实时/日/周/月榜单
- 显示榜单标题、封面、跳转链接
- 每次打开榜单页面重新爬取
- 榜单数据不持久化

### 9. 主题切换
- 浅色主题：白色背景 + 粉色 (#FF8FB1) 强调色
- 深色主题：黑色背景 + 紫色 (#9B59B6) 强调色
- 支持手动切换和跟随系统两种模式
- 设置保存到数据库

### 10. 背景图片
- 支持上传本地图片作为背景
- 两个主题都可以设置
- 图片保存到应用数据目录

## 页面设计

### 主页（媒体库）
- 左侧 240px 侧边栏：分类、工具、设置
- 右侧视频网格：16:9 横版卡片
- 顶部搜索框 + 统计信息

### 详情/编辑页
- 左侧封面 + 缩略图条
- 右侧信息表单（查看/编辑模式切换）
- 底部固定保存按钮

### 榜单页面
- 侧边栏入口
- 多个榜单标签页（实时/日/周/月）
- 卡片式榜单列表

## UI 风格
- 浅色模式：背景 #F8F8FA，白色侧边栏，粉色强调
- 深色模式：背景 #1A1A2E，深色侧边栏，紫色强调
- 圆角 16px，柔和阴影
- Noto Sans SC 字体
